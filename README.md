# SDK Packager

A packager to generate the sdk tar.gz files that go on Cloudflare.

#### Clone this repo
```shell
git clone --recurse-submodules https://github.com/coredevices/sdk-packager
```

# Usage

> Build an sdk
```shell
./pack-sdk.sh $VERSION
```

Run locally
```shell
python3 -m http.server -d dist/
```
